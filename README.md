# lazycreatives.github.io

The **Lazy Creatives** homepage **and** brand source of truth → https://lazycreatives.github.io

Lazy Creatives is a family of desktop tools that take the boring, behind-the-scenes work
of making music off your plate. *Looks lazy. Works obsessively.*

## In this repo
- `index.html` — the homepage (links to each tool)
- `brand-guide.html` — the **entity brand sheet**: palette, type, voice, logo usage, artist brief
- `logo.png` — primary logo (transparent) · `ref-sloth.png` — character crop for the brief
- `brand-src/` — master art (`LC LOGO.png`) + colour swatches
- Drop the **licensed** Slothy Jelly webfont at `slothy-jelly.woff2` to render the display font
  (it's free for personal use only — buy a commercial licence before shipping).

## Tools
- [Backups](https://github.com/LazyCreatives/lazycreatives-backups) — verified backups for DAW projects
- [Uploader](https://github.com/LazyCreatives/lazycreatives-uploader) — auto-publish & manage SoundCloud uploads
